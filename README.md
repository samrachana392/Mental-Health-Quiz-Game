# Mental Health QUIZ Game
A simple command line based quiz game developed using python and pandas . THe quiz reads questions from a CSV file, displays multiple choice questions, checks user answers and shows the final score.

## Features
- Reads quiz questions from a CSV file
- Multiple-choice questions (A, B, C, D)
- Random selection of questions
- Prevents repeating the same question
- Validates user input
- Displays final score at the end

## Technologies used
- Python
- Pandas
- Random module
